TRABAJO SOBRE PERIFÉRICOS: Cintas Magnéticas y Micrófono

Cintas Magnéticas:

Las cintas magnéticas son periféricos de almacenamiento que guardan información mediante magnetización sobre una cinta plástica. Se utilizan principalmente para copias de seguridad y almacenamiento a largo plazo.

Micrófono

El micrófono es un periférico de entrada que convierte el sonido en señales eléctricas que el ordenador puede procesar.
