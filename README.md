TRABAJO SOBRE PERIFÉRICOS: Cintas Magnéticas y Micrófono

Cintas Magnéticas:

Las cintas magnéticas son periféricos de almacenamiento que guardan información mediante magnetización sobre una cinta plástica. Se utilizan principalmente para copias de seguridad y almacenamiento a largo plazo.

Micrófono

El micrófono es un periférico de entrada que convierte el sonido en señales eléctricas que el ordenador puede procesar.

5. Conclusiones

Las _cintas magnéticas_ siguen siendo una solución eficaz para el almacenamiento masivo y copias de seguridad, especialmente en entornos profesionales.
El _micrófono_, por su parte, es un periférico fundamental para la comunicación, educación y creación de contenido, siendo ampliamente utilizado tanto a nivel doméstico como profesional.
Ambos periféricos cumplen funciones esenciales dentro de los sistemas informáticos.
